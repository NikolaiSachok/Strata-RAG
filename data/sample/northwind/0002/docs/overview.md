# Orbit Focus — Overview

**Category:** productivity / focus timer
**Theme:** space, astronomy, night sky
**Mascot:** none
**Has humor:** no — calm and minimalist by design

## What it is
A Pomodoro-style focus timer themed around spaceflight. Each completed focus block
advances a spacecraft along an orbit; weekly progress is shown as a growing
constellation.

## Key screens
- Timer (the launch / focus session)
- Orbit map (sessions completed today)
- Constellation (weekly focused-hours summary)

## Theme notes
Strictly astronomical: orbits, stations, refuelling, constellations. The mood is quiet
and serious; there is intentionally no humor or mascot.
